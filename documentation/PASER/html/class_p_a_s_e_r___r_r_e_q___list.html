Detailed Description

Class represents an entry in the RREQ list.

This class is an implementation of the RREQ list. Here, we maintain a map of the RREQs the RREPs of which have not been received yet. Besides, a map of the RREPs the RREP-ACKs of which have not been received yet is also maintained in this class.
